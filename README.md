# ShareIt — сервис совместного использования вещей

ShareIt — это веб‑сервис для шеринга (совместного использования) вещей между пользователями. Платформа позволяет:
* делиться своими вещами с другими;
* искать и арендовать нужные предметы на определённый срок;
* оставлять запросы на вещи, которых пока нет в сервисе;
* получать ответы на запросы от других пользователей;
* оставлять отзывы после аренды.

## Функциональные возможности

### Основные функции

* **Добавление и управление вещами** (Item):
    * добавление новой вещи с названием, описанием и статусом доступности;
    * редактирование информации о вещи (только владельцем);
    * просмотр списка своих вещей;
    * поиск вещей по названию или описанию (только доступных для аренды).

* **Бронирование вещей** (Booking):
    * создание запроса на бронирование с указанием дат;
    * подтверждение или отклонение бронирования владельцем вещи;
    * просмотр информации о конкретном бронировании;
    * просмотр списка бронирований с фильтрацией по статусу (все, текущие, завершённые, будущие, ожидающие подтверждения, отклонённые).

* **Отзывы** (Comment):
    * оставление отзыва после возврата вещи;
    * проверка, что пользователь действительно арендовал вещь перед оставлением отзыва;
    * просмотр отзывов о вещи.

* **Запросы на вещи** (ItemRequest):
    * создание запроса, если нужной вещи нет в сервисе;
    * просмотр своих запросов и ответов на них;
    * просмотр запросов других пользователей;
    * добавление вещи в ответ на запрос.

* **Валидация и шлюз** (Gateway):
    * разделение приложения на два модуля: `shareIt-server` (основная логика) и `shareIt-gateway` (валидация и приём запросов);
    * валидация входных данных на стороне шлюза для снижения нагрузки на основное приложение;
    * взаимодействие между модулями через REST‑запросы.

## Архитектура и технологии

### Технологический стек

* **Backend**: Spring Boot, Spring MVC, Spring Data JPA, Hibernate
* **База данных**: PostgreSQL
* **Сборка**: Maven (многомодульный проект)
* **Тестирование**: JUnit 5, MockMVC, @JsonTest
* **Взаимодействие с API**: REST
* **Документация API**: Postman-коллекция (для ручного тестирования)
